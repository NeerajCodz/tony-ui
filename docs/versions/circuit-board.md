# Circuit Board

## Description
Complex, technical aesthetic featuring PCB-like traces and nodal connections.

## Key Visual Characteristics
- **Shape:** Rectangular container with decorative "trace" extensions.
- **Borders:** Interrupted lines ending in small circles (nodes).
- **Glow/Lighting:** "Signal" pulses traveling along the border lines.
- **Animations:** Sequential lighting of border nodes.

## CSS Variables / Tokens
- `--node-size`: `4px`
- `--trace-width`: `1px`

## Best Use Cases
- Upgrade menus
- System diagnostics
- Tech-tree visualizations
