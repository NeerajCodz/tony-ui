# Version: Data Panel

## Primary Shape: Asymmetric Trapezium
A rectangle with one or two specific corners cut at a shallow angle or large chamfer to create a data-tab look.

## Geometry Specs
- **Corner Style**: Asymmetric. Top-Right is cut `20px` diagonal. All others sharp.
- **Clip Path**: `polygon(0 0, calc(100% - 20px) 0, 100% 20px, 100% 100%, 0 100%)`
- **Decoration**: Often includes a header line or status bar in the cut area.

## Design Philosophy
Information-heavy. Looks like a file folder or a specific data readout panel on a spaceship dashboard.
