# Data Panel

## Description
Dense, information-heavy panel designed for complex telemetry and data readouts.

## Key Visual Characteristics
- **Shape:** Rectangular with a specialized "header" tab.
- **Borders:** Thin, high-contrast lines with corner accents.
- **Glow/Lighting:** Low ambient glow, high-contrast text.
- **Animations:** Scanning line effect that periodically passes over the panel.

## CSS Variables / Tokens
- `--header-height`: `24px`
- `--accent-opacity`: `0.8`

## Best Use Cases
- Telemetry displays
- Inventory management
- Log consoles
