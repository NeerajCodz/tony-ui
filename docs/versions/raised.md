# Version: Raised

## Primary Shape: Elevated Rounded Rectangle
A standard shape that uses shadow and lift to simulate physical depth.

## Geometry Specs
- **Corner Style**: Rounded (`6px`).
- **Shadow**: `box-shadow: 4px 4px 0px var(--border-color)` (Hard shadow) or large soft shadow.
- **Transform**: `translateY(-2px)`.

## Design Philosophy
Tactile. Provides clear affordance that the element is clickable or separate from the background.
